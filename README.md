# Avatar Project

This repository contains the source code for the Avatar project, a comprehensive system combining AI agents, real-time communication, and augmented reality features.

## Project Structure

The project is organized into several key components:

### [FrontFace](./FrontFace)
The frontend application built with **Next.js**. It serves as the user interface for interacting with the avatar and other system features.
- **Tech Stack**: Next.js, React, Tailwind CSS, LiveKit Components.

### [HumanAI](./HumanAI)
The intelligent agent backend powered by **Python**. This component handles the logic for the AI avatar, including conversation management and instructions.
- **Tech Stack**: Python, LiveKit Agents.

### [RAG-main](./RAG-main)
A Retrieval-Augmented Generation (RAG) system. This module provides the knowledge base and retrieval capabilities to enhance the AI's responses with relevant information.
- **Tech Stack**: Python, Vector Database (likely Chroma or similar).

### [AR](./AR)
An Augmented Reality and computer vision module. It includes features for hand gesture recognition and other visual interactions.
- **Tech Stack**: Python, MediaPipe.
